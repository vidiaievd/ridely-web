# Ridely Web

React web application for the Ridely platform.

## 🚀 Quick start

### Prerequisites
- Node.js 18+
- npm or yarn

### Setup and run

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

App is available at: http://localhost:5173

## 🛠️ Technologies

- **React 18** + TypeScript
- **Vite** - build tool
- **Zustand** - state management
- **React Query** - API requests
- **Mapbox GL JS** - maps
- **Tailwind CSS** - styles
- **React Router** - routing

## 📁 Project structure

```
src/
├── components/        # Reusable components
├── pages/             # App pages
├── hooks/             # Custom hooks
├── stores/            # Zustand stores
├── services/          # API services
├── types/             # TypeScript types
├── utils/             # Utilities
└── assets/            # Static assets
```

## 🎯 Main features

- **Spot map** with level filtering
- **User authentication**
- **Group management** for shared surfing
- **Weather data** for spots
- **Responsive design** for mobile
